# Frontend and Backend Nuve - Tubes LKMO 06 - Website Fashion

## Step to collab this project:

### 1. Backend
- Setup:
```
git clone <link_repo>
cd nuve-be
npm install
```
- Add `.env` and input API Key Gemini from Gemini API
```
GEMINI_API_KEY=<api_key>
PORT=8080
```


### 2. Frontend
```
git clone <link_repo>
cd nuve-fe
```
